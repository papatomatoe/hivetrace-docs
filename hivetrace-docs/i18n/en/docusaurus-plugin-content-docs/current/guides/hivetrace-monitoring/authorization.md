## Authorization

To get started, open the web interface of your HiveTrace instance and authenticate using the credentials provided by your system administrators.  

- For on-premise deployments, access is typically granted by your organization’s DevOps engineer.  
- If you are using an instance provisioned by the HiveTrace team, the credentials are shared during the onboarding and initial service setup process.
