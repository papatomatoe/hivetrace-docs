## Dashboard

After authentication, you are presented with the dashboard, which provides a consolidated view of system activity. It offers immediate visibility into key metrics, helping teams quickly assess the current state of their AI infrastructure.

![Dashboard](/img/dashboard.png)

The dashboard displays:
- total message volume and activity over the last 24 hours;
- the overall number of detected violations and their daily trend;
- token usage statistics, including total consumption, tokens used today, and the average number of tokens per request;
- data on violations related to token overuse;
- recent alerts to support timely response to potential incidents.

The dashboard enables faster anomaly detection, better capacity oversight, and helps maintain secure and reliable AI operations.
