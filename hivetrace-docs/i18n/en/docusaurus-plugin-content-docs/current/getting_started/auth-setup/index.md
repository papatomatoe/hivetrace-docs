# Authorization

This section describes the first login process to the HiveTrace platform after deployment, as well as subsequent authorization.

## First authorization

When logging in for the first time after deployment, use the administrator credentials configured during deployment.

**Steps for the first login:**

1. Open the HiveTrace web interface in your browser
2. Enter the administrator login and password
3. After successful authorization you will get access to the admin panel

## Subsequent authorization

For subsequent logins, use the username and password provided by your system administrator.

::::note Important
- The administrator must register you in the system
- User credentials are created and managed via the [Team Setup](../team-setup/index.md) section
- Keep your credentials secure and do not share them with third parties
::::

## Access recovery

If you forgot your password or cannot log in, contact your administrator.
