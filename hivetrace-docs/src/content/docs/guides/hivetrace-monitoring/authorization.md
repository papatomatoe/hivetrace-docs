Чтобы приступить к работе, откройте веб-интерфейс вашего инстанса HiveTrace и выполните аутентификацию с использованием учетных данных, предоставленных администраторами системы.

- При on-premise развертывании доступ обычно предоставляет ответственный DevOps-инженер вашей организации.
- Если вы используете инстанс, развернутый командой HiveTrace, учетные данные передаются в рамках процесса подключения и первичной настройки сервиса.
