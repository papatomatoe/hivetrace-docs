Страница **«API-токены»** предназначена для управления токенами авторизации, используемыми при интеграции с HiveTrace. В этом разделе вы можете просматривать список созданных токенов, а также создавать и удалять их при необходимости.

![Страница "API-Токены"](../../../../assets/img/api_tokens_page.webp)

## Создание API-токена

Чтобы создать новый токен, нажмите **«Добавить новый API-токен»** и укажите его **название**. После создания откроется окно со значением токена.

> **Важно:** токен отображается **только один раз**. Рекомендуется сразу сохранить его в защищенном месте, поскольку повторный просмотр невозможен.

Созданный токен используется для аутентификации при работе с HiveTrace — включая интеграцию через **SDK**, **API** и **прокси**.
