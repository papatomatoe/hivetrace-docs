HiveTrace предоставляет серверные API для мониторинга, валидации и модерации взаимодействий с LLM. Платформа анализирует как входящие сообщения пользователей, так и исходящие ответы моделей с использованием настраиваемых модулей проверки.

HiveTrace может быть встроен в пайплайн приложения для применения контролей безопасности, обнаружения чувствительных данных, проверки политик и сохранения результатов мониторинга до передачи данных в модель или конечному пользователю.

**Поддерживаемые модули:**

- **Dataclean** — обнаружение и маскирование/очистка чувствительных данных
- **Guardrails** — проверка встроенных политик
- **Custom Policies** — пользовательские правила валидации
- Учет токенов
- Связывание сессий и пользователей
- Сохранение результатов мониторинга
