Все запросы требуют следующий заголовок:

```
Authorization: Bearer <API_TOKEN>
```

| Сценарий          | Результат            |
| ----------------- | -------------------- |
| Токен отсутствует | **401 Unauthorized** |
| Токен неверный    | **401 Unauthorized** |

API-токены создаются и управляются через HiveTrace Web UI.
