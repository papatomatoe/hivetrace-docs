## Application

A logical unit that represents your LLM app in HiveTrace. The SDK uses `application_id` to associate monitoring events.

## Guardrail

A built-in baseline protection layer with predefined rules.

## Custom Policies

Configurable rules (often expressed as system-level instructions) that reflect business constraints and compliance requirements.

## Monitoring Result

The processing output: violation flags, token counters, and other signals used for alerts and investigations.
