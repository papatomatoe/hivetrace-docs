# Ключевые понятия

## Application

Логическая единица, которая соответствует вашему LLM‑приложению в HiveTrace. `application_id` используется SDK для привязки событий мониторинга.

## Guardrail

Встроенный слой базовой защиты с преднастроенными правилами.

## Custom policies

Настраиваемые правила (обычно в виде системного промпта/инструкций), которые отражают бизнес‑ограничения и требования комплаенса.

## Monitoring result

Результат обработки события: флаги нарушений, счётчики токенов и другие метрики, которые используются для алертов и расследований.
