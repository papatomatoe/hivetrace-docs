## Дашборд

После авторизации открывается стартовая страница с дашбордом, содержащим сводную информацию о работе системы. Он предоставляет быстрый обзор ключевых метрик и помогает оперативно оценивать текущее состояние AI-инфраструктуры.

![Дашборд](/img/dashboard.png)

На дашборде отображаются:
- общее количество сообщений и активность за последние сутки;
- общее число выявленных нарушений и их динамика за день;
- статистика потребления токенов, включая суммарное значение, использование за текущие сутки и среднее количество токенов на запрос;
- данные о нарушениях, связанных с превышением лимитов потребления;
- информация о последних оповещениях для своевременного реагирования на потенциальные инциденты.

Использование дашборда позволяет быстро выявлять аномалии, контролировать нагрузку и поддерживать стабильную и безопасную работу AI-приложений.
