## Пользователи

Страница **«Пользователи»** содержит список пользователей вашего AI-приложения (например, чат-бота), которые взаимодействуют с моделью и отправляют сообщения. Раздел предназначен для централизованного управления пользователями и анализа их активности.

![Страница "Пользователи"](/img/users_page.png)

В этом разделе вы можете:

* просматривать список пользователей;
* редактировать и удалять существующие записи;
* добавлять пользователей вручную.

Пользователи также могут добавляться **автоматически**. Если при отправке запроса в HiveTrace вы укажете новый идентификатор пользователя:

* в **additional parameters** при использовании API или SDK, либо
* в **HTTP-заголовках** при работе через прокси,

такой пользователь появится в списке автоматически. В этом случае запись будет содержать только **ID пользователя** — имя при необходимости добавляется вручную через интерфейс.

### Детали пользователя

При нажатии на идентификатор пользователя открывается страница с подробной информацией о его активности. **По умолчанию открывается вкладка «Сессии»**, расположенная рядом со вкладкой **«Файлы»**.

![Сессии Пользователя](/img/users_sessions.png)

Вкладка **«Сессии»** отображает историю взаимодействий пользователя с приложением и позволяет анализировать отдельные диалоги.

Переключившись на вкладку **«Файлы»**, вы увидите список всех файлов, которые пользователь загружал в процессе общения с ботом.

![Файлы Пользователя](/img/users_files.png)

> **Важно:** содержимое пользовательских файлов **не проходит автоматическую цензуру и анализ**. Файлы доступны исключительно для ручного просмотра и скачивания и не участвуют в механизмах мониторинга и политик безопасности.
